MercadoLibre Publisher

MercadoLibre Publisher / Connector for Odoo.

What it is?

MercadoLibre Publisher (meli_oerp) by Moldeo Interactive is a MercadoLibre connector for Odoo that allows syncronize products with images and attributes, get mercadolibre orders, get mercadolibre questions and more features.

Features

Massive publication from Odoo to MercadoLibre / Import all products from MercadoLibre to Odoo
Get Orders with all shipping and client information from MercadoLibre to Odoo
Get Customer's Questions from MercadoLibre to Odoo
Configure access with your credentials
Configure basic template information for massive publications
Configure all the parameters to keep prices, stocks synchronized in the best interest of your business model.
MercadoLibre Publisher is the base core of a suite of MercadoLibre extensions
